Для запуска без виртуальной машины:

Для запуска проекта, необходимо открыть файл VB_no.env в папке srcs и внизу в файле в строках с переменными ADM_DEVICE, DB_DEVICE, RESUME_DEVICE, WP_DEVICE
в части пути "/home/user/Desktop/github/(продолжение пути до папки)" заменить часть 
             "/home/user/Desktop/github/" на путь на Вашей машине до проекта Inception.
